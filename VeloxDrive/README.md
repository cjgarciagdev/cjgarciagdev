#  VeloxDrive: Landing Page Automotriz Premium

![UI/UX](https://img.shields.io/badge/UI%2FUX-Premium-FFD700)
![Responsive](https://img.shields.io/badge/Responsive-Mobile--First-blue)
![Tech](https://img.shields.io/badge/Tech-HTML5_--_CSS3-orange)

**VeloxDrive** es una propuesta de diseño web para la industria automotriz de lujo, enfocada en la elegancia visual y una experiencia de usuario (UX) fluida. El proyecto demuestra habilidades avanzadas en maquetación con CSS moderno sin dependencia de frameworks externos.

---

##  Características Técnicas y de Diseño

### 1. Estética Visual de Alto Impacto
*   **Hero Section Dinámico:** Uso de imágenes de alta resolución optimizadas para carga rápida.
*   **Glastmorfismo:** Aplicación de efectos de transparencia y desenfoque (backdrop-filter) en menús y tarjetas para un look moderno.
*   **Paleta de Colores Curada:** Contraste sofisticado basado en el concepto "Dark Mode" para resaltar los vehículos.

### 2. Animaciones Interactivas
*   **Micro-interacciones:** Hover effects suaves en botones de llamada a la acción (CTA) y elementos de navegación.
*   **Transiciones Fluidas:** Navegación entre secciones con scroll suave y efectos de entrada al hacer scroll.

### 3. Mobile-First Responsive Design
*   **Maquetación con Flexbox/Grid:** Estructura completamente flexible que se adapta desde smartphones de 320px hasta monitores ultrawide.
*   **Assets Optimizados:** Imágenes y logotipos preparados para pantallas de alta densidad (Retina Displays).

---

##  Tecnologías

*   **HTML5 Semántico:** Para una mejor SEO y accesibilidad.
*   **CSS3 Avanzado:** Uso de variables CSS (Custom Properties), Keyframes para animaciones y Media Queries para el diseño responsivo.
*   **Vanilla JS (Opcional):** Lógica mínima para el manejo del menú móvil y efectos de scroll.

---

##  Estructura del Proyecto

*   `index.html`: Estructura semántica del sitio.
*   `style.css`: Hoja de estilos principal con el sistema de diseño completo.
*   `img/`: Directorio de recursos visuales optimizados.
