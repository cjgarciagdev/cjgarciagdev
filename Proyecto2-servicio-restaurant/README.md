#  Landing Page: Servicio de Restaurant

Una página web profesional diseñada para establecimientos gastronómicos que buscan tener presencia digital rápida y efectiva.

---

##  Características
- **Menú Visual:** Sección dedicada a mostrar los platos principales con precios y descripciones.
- **Información de Contacto:** Integración clara de ubicación y horarios.
- **Diseño Temático:** Estilos CSS personalizados para transmitir la atmósfera del restaurante.

##  Tecnologías
- **HTML5**
- **CSS3** (Hoja de estilos personalizada).
