## 🎨 Tipo de Comentarios Agregados

### 1. Comentarios de Bloque (/** */)
Usados para:
- Documentación de componentes
- Explicación de secciones importantes
- Documentación JSDoc de interfaces

Ejemplo:
```typescript
/**
 * COMPONENTE LOGIN
 * Pantalla de inicio de sesión del sistema hospitalario
 * Valida las credenciales del usuario antes de permitir el acceso
 */
```

### 2. Comentarios de Línea (//)
Usados para:
- Explicación de variables
- Descripción de funciones
- Aclaraciones de lógica

Ejemplo:
```typescript
// Almacena el nombre del usuario actualmente logueado (null si no hay sesión)
const [usuario, setUsuario] = useState<string | null>(null);
```

### 3. Comentarios JSX ({/* */})
Usados para:
- Explicación de elementos JSX
- Descripción de secciones del UI

Ejemplo:
```jsx
{/* Campo de entrada para el nombre de usuario */}
<input type="text" placeholder="Usuario" />
```

### 4. Comentarios JSDoc
Usados para:
- Documentación de interfaces TypeScript
- Descripción de propiedades
- Ejemplos de uso

Ejemplo:
```typescript
/** 
 * Fecha de vencimiento del producto
 * Formato: YYYY-MM-DD
 */
vencimiento: string;
```
